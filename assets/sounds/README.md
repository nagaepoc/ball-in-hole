# Sound Assets

Placeholder for game sounds. In a real deployment, you would need:

1. `roll.mp3` - Ball rolling sound
2. `win.mp3` - Level complete sound
3. `bump.mp3` - Wall collision sound

Sound requirements:
- Short, optimized audio files
- Properly licensed or created sounds
- Multiple formats for browser compatibility (MP3, OGG, WAV)
- Volume levels balanced for gameplay
